# Netflix Data Analysis

Exploratory Data Analysis of Netflix movies from the 1990s.

## Project Overview

This project analyzes Netflix movie data to explore patterns in movie duration during the 1990s.

## Dataset

The dataset includes information about Netflix titles such as:

- Title
- Release year
- Duration
- Genre

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

## Project Files

| File | Description |
|-----|-----|
| netflix_data.csv | Dataset |
| notebook.ipynb | Data analysis notebook |
| redpopcorn.jpg | Visualization image |

## Key Questions

- What was the most common movie duration in the 1990s?
- How did movie durations vary across the decade?
